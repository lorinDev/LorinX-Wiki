# LoRA: Low-Rank Adaptation of Large Language Models

- **Authors**: Edward J. Hu, Yelong Shen, Phillip Wallis, Zeyuan Allen-Zhu, Yuanzhi Li, Shean Wang, Lu Wang, Weizhu Chen
- **Published**: June 2021 (v2: Oct 2021)
- **Venue**: arXiv (2106.09685)
- **Code**: https://github.com/microsoft/LoRA

## One-Liner
Proposes freezing pre-trained weights and injecting trainable low-rank decomposition matrices, reducing trainable parameters 10,000× for GPT-3 175B with no inference latency overhead.

## Key Results
- GPT-3 175B: trainable params reduced 10,000×, GPU memory 3× less
- On-par or better than full fine-tuning on RoBERTa, DeBERTa, GPT-2, GPT-3
- Empirical evidence for rank-deficiency in model adaptation
