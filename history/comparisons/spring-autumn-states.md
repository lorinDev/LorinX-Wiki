## 崛起与衰落模式

### 模式对比

```
齐：改革(管仲)→霸业→内乱→田氏代齐→灭亡
晋：扩张(献公)→霸业(文公)→卿族坐大→三家分晋
楚：开拓(武王/庄王)→称霸→保守→屡败于秦→灭亡
秦：西陲→变法(商鞅)→制度碾压→统一天下
吴：崛起(阖闾)→破楚→北上争霸→被越消灭
越：卧薪尝胆→灭吴→短暂霸权→内乱→灭亡
鲁：周礼正统→三桓专政→文化贡献→军力弱小→灭亡
```

## 历史规律

1. **制度决定论**：改革越彻底的（秦），最终胜出
2. **贵族制 vs 官僚制**：晋国卿族坐大分裂；秦国瓦解贵族、建立官僚体系
3. **地理位置**：四塞之国的秦最安全；平原之地的中原诸国互相消耗
4. **文化软实力**：鲁虽小但影响最大——思想比疆域更持久
