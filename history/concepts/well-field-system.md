# 井田制

## 核心定义

井田制是西周至春秋时期的土地制度，得名于将一块土地按"**井**"字形划分为九块。中间为"公田"（产出归领主），周围八块为"私田"（产出归农户）。理想化的井田制反映了**土地公有**的理念。

## 基本结构

```
   私田  私田  私田
   私田 [公田] 私田
   私田  私田  私田
```

- **公田**（中心）：八户共同耕种，产出归领主/国家
- **私田**（四周）：八户各自耕种，产出归自己
- 一井 = 900 亩（约合今 182 亩）
- 八家为一井，构成基层生产单位

## 真实性争议

- **孟子**最早系统描述井田制（"方里而井，井九百亩"）
- 但孟子自己也承认"其详不可得闻也"
- 现代学者多认为：
  - 可能有土地**定期分配**的实践
  - 但"井"字形九宫格划分是理想化的构想
  - 井田制可能是儒家对"理想社会"的一种**托古改制**

## 与[[zhou-feudal-system|封建制]]的关系

- 封建制是政治制度（分封），井田制是经济基础（土地分配）
- 封建等级与土地分配对应：天子→诸侯→卿大夫，每级都控制一定范围的土地
- "普天之下，莫非王土"——名义上所有土地归天子所有

## 瓦解原因

春秋战国时期，井田制逐渐瓦解：

1. **铁器和牛耕**的普及 → 生产力提高 → 个体农户能耕种更多土地 → 集体耕作不必要
2. **人口增长** → 人均土地减少 → 井田分配无法维持
3. **土地私有化** → 贵族和富农购买土地 → 打破公田私田的界限
4. **战争需求** → 各国需要更多税收 → 按实际土地面积征税 → 废除井田制

### 历史节点

- 前 594 年，鲁国"**初税亩**"——按土地面积征税，正式承认土地私有（井田制瓦解的标志）
- 前 350 年，商鞅"**废井田，开阡陌**"——从法律上彻底废除井田制

## 后世影响

- 井田制成为后世**土地改革的理想模板**
- 王莽"王田制"、均田制、太平天国"天朝田亩制度"都受其启发
- 作为儒家理想社会的象征，影响了中国两千年的土地思想
