# 周代封建制

## 核心定义

周代封建制是周朝建立的一套**分封治理体系**：天子将土地和人民分封给同姓宗亲和功臣，建立诸侯国；诸侯国内又层层分封。它与宗法制结合，形成"家国同构"的政治架构。

## 与欧洲 Feudalism 的异同

| 维度 | 周代封建制 | 欧洲 Feudalism |
|------|-----------|----------------|
| 纽带 | **血缘** + 契约 | **契约**为主 |
| 继承 | 嫡长子继承（宗法制） | 长子继承（各地不同） |
| 层级 | 天子→诸侯→卿大夫→士 | 国王→公爵→伯爵→骑士 |
| 经济基础 | 井田制（土地公有框架） | 庄园制（领主私有） |
| 瓦解原因 | 郡县制替代（内因） | 商品经济 + 王权强化 |

## 分封体系

### 诸侯类型
- **同姓诸侯**：姬姓子弟（如鲁、晋、燕、卫）
- **功臣诸侯**：如太公望（姜尚）封于齐
- **前朝后裔**：如宋（商朝后裔）、陈（舜后裔）

### 诸侯义务
1. **朝贡**：定期朝见天子，进贡方物
2. **从征**：天子征伐时提供军队
3. **筑城**：参与王都建设
4. **述职**：定期汇报治下情况

### 天子权力
- 分封权：册命诸侯，授予土地人民
- 废立权：对不守规矩的诸侯可讨伐、废黜
- 巡狩权：定期巡察各诸侯国

## 宗法制与封建制的结合

宗法制解决"谁继承"的问题（嫡长子），封建制解决"分什么"的问题（土地、人民、权力）。二者结合形成：

```
天子（大宗）──诸侯（小宗/大宗）──卿大夫（小宗/大宗）──士
  │              │                    │
  王畿           封国                 采邑
```

"大宗百世不迁，小宗五世则迁"——大宗永远保持宗主地位，小宗五代后可自立门户。

## 瓦解过程

### 春秋：封建制的松动
- 诸侯不朝天子，"礼乐征伐自诸侯出"
- 诸侯国内卿大夫专权（如鲁国三桓、晋国六卿）
- 县制的出现削弱了封建层级
- 楚、秦等国直接设县，由国君委派长官

### 战国：封建制的崩溃
- 各国变法废除世卿世禄
- 商鞅变法："宗室非有军功论，不得为属籍"
- 郡县制全面推广
- 官僚制取代贵族制

## 历史遗产

- 分封制在汉初一度复活（郡国并行），引发七国之乱
- 此后历代虽有封王，但**实封不治民**，封建制在政体上已终结
- "封建"一词成为传统社会的代名词（"封建社会"）
