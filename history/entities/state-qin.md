# 秦国 (Qin)

## 概述

秦国起源于陇西（今甘肃东部），早期为周王室的附庸，因护送周平王东迁有功正式封为诸侯（前 770 BCE）。在春秋时期被中原诸侯视为"西戎"，但逐步崛起，至战国时期成为最强国家，最终于前 221 年统一六国。

## 地理

早期偏处西陲（今甘肃天水→陕西关中），"僻在雍州，不与中国诸侯之会盟"。夺得周故地岐、丰后拥有沃野千里的关中平原，进可攻退可守。

## 春秋时期

### 秦穆公称霸西戎
- 秦穆公（前 659–621 BCE）：春秋五霸之一（有争议）
- 任用百里奚（"五羖大夫"）、蹇叔等贤臣
- "益国十二，开地千里，遂霸西戎"
- **崤之战**（前 627 BCE）：因企图东进中原遭晋伏击惨败，此后长期被晋封锁
- 秦晋之好：与晋国建立婚姻联盟（但屡次破裂）

### 春秋中后期
- 秦晋长期在河西（黄河以西）拉锯
- 国内政治相对稳定，公族势力有限
- 不被中原主流文化接纳，但也因此较少卷入中原纷争

## 战国时期：从偏安到最强

### 秦孝公与商鞅变法（前 356 BCE）
这是中国古代**最彻底、影响最深远**的变法：

| 领域 | 改革内容 |
|------|---------|
| **政治** | 推行县制；编定户籍，什伍连坐；统一度量衡 |
| **军事** | 二十等爵制，**以军功授爵**——"宗室非有军功，不得为属籍" |
| **经济** | "废井田，开阡陌"——正式承认土地私有；重农抑商 |
| **社会** | 分户令，强制成年男子分家；移风易俗 |

> 商鞅变法的结果是"秦民大悦，道不拾遗，山无盗贼，家给人足"——使秦国从落后边陲变为效率极高的战争机器。

### 统一之路
- 司马错取巴蜀（前 316 BCE）：获得战略纵深和资源基地
- 范雎提出"**远交近攻**"战略
- 白起伊阙之战（前 293 BCE）、长平之战（前 260 BCE）奠定军事霸权
- 嬴政（前 247 BCE 即位）在李斯辅佐下发动统一战争：
  - 灭韩（前 230）→ 灭赵（前 228）→ 灭魏（前 225）→ 灭楚（前 223）→ 灭燕（前 222）→ 灭齐（前 221）

## 成功原因

1. **地理位置**：关中四塞之地，易守难攻
2. **制度优势**：商鞅变法最彻底，建立了高效的国家机器
3. **人才引进**：大量使用外来人才（百里奚、商鞅、张仪、范雎、李斯均为"外国人"）
4. **战略得当**：远交近攻，逐个击破
