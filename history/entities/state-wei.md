# 魏国 (Wei)

## 概述

魏国是[[state-jin|三家分晋]]后最强的一支，**战国初期首霸**，在李悝变法和吴起军改的推动下率先崛起。但因战略失误和人才流失（商鞅、张仪、范雎等均从魏入秦），逐渐衰落，最终于前 225 年被秦所灭，历时 178 年。

## 地理

都城先在**安邑**（今山西夏县），后因受秦压迫，迁至**大梁**（今河南开封）。核心区在黄河中下游（今河南东部、山西南部）。疆域被韩、赵隔断，呈不连续形态，战略防御困难。

## 崛起：战国首霸

### 魏文侯（前 445–前 396 BCE）
- 战国初期最具眼光的君主
- **尊贤重士**：师事子夏（孔子弟子），礼遇田子方、段干木
- 开战国变法先河

### 李悝变法（战国最早变法）
| 领域 | 内容 |
|------|------|
| **政治** | 废除世卿世禄，选贤任能 |
| **经济** | "尽地力之教"——精耕细作；**平籴法**——国家调控粮价 |
| **法律** | 著《**法经**》（六篇），中国第一部成文法典，商鞅携之入秦 |
| **军事** | 吴起改革军制，建立**魏武卒**（精锐重装步兵） |

### 吴起与魏武卒
- 吴起任西河守，创建精锐部队**魏武卒**
- 士兵选拔严苛："衣三属之甲，操十二石之弩"
- 与诸侯大战七十六，全胜六十四
- **前 389 年阴晋之战**：吴起以五万魏武卒大破秦军五十万
- 魏武卒的战绩使魏国军事霸权达到顶峰

## 转折：从首霸到衰落

### 魏惠王（前 369–前 319 BCE）
- 在位 50 年，前期延续霸业，后期急剧衰退
- **逢泽之会**（前 344 BCE）：率十二诸侯朝见周天子，霸业巅峰
- 但两大战略失误葬送霸权：

| 失误 | 后果 |
|------|------|
| 不杀商鞅 | 商鞅入秦，推动秦国崛起 |
| 攻赵伐韩 | 引发桂陵、马陵之战 |

### 桂陵之战（前 354 BCE）
- 魏攻赵都邯郸 → 齐将**孙膑**"**围魏救赵**"
- 魏将庞涓大败，但未伤根本

### 马陵之战（前 342 BCE）——魏国由盛转衰的转折点
- 魏攻韩 → 孙膑再出，以**减灶计**诱庞涓追击
- 庞涓至马陵道，见树上书"**庞涓死于此树之下**"
- 齐军万弩齐发，魏军覆没，庞涓**自杀**
- 魏国精锐尽丧，从此失去霸主地位

## 人才流失

魏国培养了战国最多的顶级人才，但全部外流：

| 人才 | 去向 | 后果 |
|------|------|------|
| **商鞅** | 入秦 | 变法使秦富强 |
| **孙膑** | 入齐 | 桂陵、马陵两次击败魏国 |
| **张仪** | 入秦 | 连横破合纵 |
| **范雎** | 入秦 | 远交近攻战略 |
| **吴起** | 入楚 | 变法强楚 |

> **致命原因**：魏国贵族势力强大，排挤外来人才；秦国则广纳天下之士。此消彼长。

## [[lord-xinling|信陵君]]的回光返照

- 前 247 年，信陵君率五国联军大败秦军——魏国最后一次辉煌
- 但魏王猜忌夺权，信陵君抑郁而终
- 魏国再无回天之力

## 灭亡

- 前 225 年：秦将王贲引黄河水灌大梁城
- 城墙崩塌，魏王假投降，魏国灭亡
- 历时 178 年，从首霸到亡国
