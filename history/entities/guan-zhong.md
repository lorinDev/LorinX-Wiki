# 管仲

## 基本信息

- **姓名**：管夷吾，字仲
- **生卒**：？–前 645 BCE
- **国籍**：齐国
- **身份**：齐国国相、改革家
- **时代**：春秋早期（齐桓公时期）
- **被尊为**："**管子**"、**法家先驱**

## 生平

### 管鲍之交
- 管仲与**鲍叔牙**为至交
- 早年一起经商，管仲分钱时多取，鲍叔牙知其家贫不以为贪
- 齐国内乱时，管仲辅公子纠，鲍叔牙辅公子小白（后来的齐桓公）
- 管仲曾射伤公子小白（箭中带钩），差点杀死他
- 齐桓公即位后，鲍叔牙力荐管仲："**君且欲霸王，非管夷吾不可**"
- 齐桓公**不计前嫌**重用管仲为相

### 管仲改革
管仲在中国历史上首次进行了系统性的国家改革：

| 领域 | 内容 |
|------|------|
| **行政** | "叁其国而伍其鄙"——分国都21乡，农村5属，加强管理 |
| **军事** | "作内政而寄军令"——军政合一，寓兵于民 |
| **经济** | "官山海"——**盐铁国家专营**的开创者；"相地而衰征"——按土地好坏征税 |
| **外交** | "**尊王攘夷**"——尊重周天子，团结诸侯抵抗戎狄 |
| **社会** | 士农工商四民分业，不许杂处——最早的职业分区制 |

### 外交成就
- 尊王攘夷策略使齐国获得道义制高点
- 召陵之盟（前656 BCE）：率八国联军迫使楚国承认周天子权威
- 葵丘之会（前651 BCE）：齐桓公霸业顶峰

## 思想贡献

### 《管子》
- 托名管仲的著作，实际成书于战国至汉代
- 内容涵盖政治、经济、军事、哲学
- **"仓廪实则知礼节，衣食足则知荣辱"**——朴素唯物主义名言
- 提出"轻重之术"——国家通过调节市场物价来控制经济

### 法家先驱
- 管仲是中国法治思想的早期实践者
- "以法治国"最早见于《管子》
- 影响了后来的[[shang-yang|商鞅]]、[[han-fei|韩非]]等法家人物

## 评价与影响

- **孔子**：高度评价管仲——"微管仲，吾其被发左衽矣"（没有管仲，我们都成了蛮夷）
- **诸葛亮**："每自比于管仲、乐毅"
- 被后世视为**改革家的典范**和**贤相的楷模**
