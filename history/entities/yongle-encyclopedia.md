# 永乐大典

## 概述

《永乐大典》是明成祖永乐年间编纂的**全世界最大的纸质百科全书**，全书 22,877 卷、11,095 册、约 3.7 亿字。它不仅是明初的文化巅峰，也是中国文献史上最重要的大型类书。可惜历经劫难，**现存不足原书的 4%**。

## 编纂

### 背景
- 永乐元年（1403 年），明成祖朱棣命解缙主持编纂
- 初名《文献大成》，一年后成书，朱棣不满意——"尚多未备"
- 永乐三年（1405 年）重启，以姚广孝、解缙、郑赐为监修
- 动**用 2,169 人**，包括学者、抄写员、校对
- **永乐六年（1408 年）**正式完成，赐名《永乐大典》

### 编排方式
- 按《洪武正韵》的**韵目**编排
- "用韵以统字，用字以系事"——按韵查字，按字找条目
- 每个条目下**原文照录**整部书或整段文字，不做删节
- 收录先秦至明初的约 **7,000–8,000 种**典籍

> 这种"原文照录"的方式使它成为一座**辑佚宝库**——许多后世失传的古籍全靠《大典》残卷才为人所知。

## 规模

| 指标 | 数据 |
|------|------|
| 总卷数 | 22,877 卷 |
| 总册数 | 11,095 册 |
| 总字数 | 约 3.7 亿字 |
| 参与人数 | 2,169 人 |
| 编纂时间 | 5 年（1403–1408） |
| 收录古籍 | 约 7,000–8,000 种 |

> 作为对比：《大英百科全书》（第 15 版）约 4,400 万字——《永乐大典》是其 **8 倍以上**。

## 正本与副本

### 正本
- 永乐年间完成的原稿，仅此一部
- 藏于南京文渊阁，后随迁都北移至北京

### 副本（嘉靖副本）
- 嘉靖三十六年（1557 年），宫中大火，正本险些焚毁
- 嘉靖帝下令**重录副本**
- 历时六年（1562–1567），由 108 名抄写员完成
- 副本的**版式、装帧与正本完全相同**

### 正本消失之谜
嘉靖副本完成后，**正本从此消失**。三大假说：

| 假说 | 依据 |
|------|------|
| 🔥 **毁于明末战火** | 李自成撤离北京时烧毁皇宫 |
| ⚰️ **殉葬永陵** | 嘉靖皇帝喜爱此书，可能随葬入永陵地宫（考古未证实） |
| 🔥 **毁于宫中火灾** | 万历或更早的宫中大火 |

正本的失踪是中国文献史上**最大的谜案之一**。

## 散佚经过

| 时间 | 事件 | 损失 |
|------|------|------|
| 嘉靖年间 | 宫中大火，正本险些被毁 | 促使重录副本 |
| 明末清初 | 战乱 | 副本开始散佚 |
| 乾隆修《四库》 | 清点发现已缺 1,000 余册 | 约 10% 已失 |
| 晚清 | **官员监守自盗**，翰林院藏书被大量偷走 | 加速流失 |
| **1900 年** | **义和团运动 + 八国联军** | 🔥 **毁灭性打击** |

### 翰林院大火（1900 年）

1900 年义和团围攻外国使馆区，位于使馆区旁的**翰林院**（藏有《永乐大典》）被战火波及。大火之后：

- 残卷被**外国使馆人员、传教士、书商**捡走
- 英国使馆人员将其运至伦敦拍卖
- 大量卷册流散至欧、美、日各国

> 这是《永乐大典》历史上最惨烈的损失——本已残缺的副本经此一劫，十不存一。

## 现存情况

**仅存约 400 余册、800 余卷——不到原书的 4%。**

## 当代努力

- **影印出版**：现存残卷已全部影印出版，可供研究
- **辑佚**：清代以来学者从残卷中辑出大量失传古籍——《旧五代史》《宋会要》《元和姓纂》等均靠《大典》得以复原
- **全球征集**：中国国家图书馆持续从海外购回散失卷册
- **数字化**：完成现存卷的数字化

## 文化意义

- 中国**类书编纂传统的巅峰**（此前有《艺文类聚》《太平御览》，此后有《古今图书集成》《四库全书》）
- **辑佚学**的基础文献——大量宋元古籍靠它保存
- 体现明初"**盛世修典**"的文化雄心
- 永乐帝迁都北京、派郑和下西洋、修《永乐大典》——三件大事构成明初的文化格局
