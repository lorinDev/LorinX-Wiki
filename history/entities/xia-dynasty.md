# 夏朝 (Xia Dynasty)

## 概述

夏朝是中国传统史书中记载的第一个世袭王朝，与商、周合称"三代"。约存在于公元前 2070 年至前 1600 年，历时约 470 年。传统上认为由大禹（禹）或其子启建立，共传 14 代 17 王（一说 13 代 16 王）。

## 历史地位

- 标志着中国从**部落联盟**向**世袭王朝**的过渡
- 开启了"家天下"的传统，取代了尧舜禹时代的"禅让制"
- 是中国青铜时代早期的重要阶段

## 历史真实性争议

- **传统观点**：《史记·夏本纪》等文献记载了完整的夏王世系
- **考古对应**：普遍认为**二里头文化**（河南偃师，约 1900–1500 BCE）可能与夏朝有关
  - 二里头遗址发现了大型宫殿基址、青铜器作坊
  - 但遗址中**未发现文字**，无法确证与文献中夏朝的对应关系
- **西方学界**：多数将夏朝视为"半传说"（semi-legendary），认为尚缺乏直接考古证据

## 关键事件与人物

| 人物 | 事迹 |
|------|------|
| **大禹**（禹） | 治水有功，受舜禅让。传说划定九州，奠定夏朝基础 |
| **启** | 禹之子，废除禅让制，开启世袭王朝 |
| **太康** | 因失政被后羿夺取政权（"太康失国"） |
| **少康** | 中兴夏朝，恢复夏后氏统治（"少康中兴"） |
| **桀** | 末代君主，以暴虐著称，被商汤推翻 |

## 灭亡

约公元前 1600 年，商部落首领汤联合各方国，在鸣条之战中击败夏桀，夏朝灭亡。商朝建立。

## 文化遗产

- 二里头文化：中国最早的广域王权国家的考古证据
- 青铜礼器制度萌芽（爵、斝等酒器）
- 可能与后来的 [[zhou-dynasty]] "夏礼" 记忆有关
