# 商鞅

## 基本信息

- **姓名**：公孙鞅，后封于商，称商鞅
- **生卒**：约前 390–前 338 BCE
- **国籍**：卫国（出生）→ 魏国（仕途起点）→ 秦国（变法）
- **身份**：法家代表人物、改革家、秦国大良造
- **时代**：战国中期（秦孝公时期）

## 生平

### 入秦
- 商鞅本是卫国公族后裔，早年仕于魏国
- 在魏相公叔痤手下任中庶子，深受赏识
- 公叔痤临终向魏惠王推荐商鞅，又建议"不用则杀之"——魏王均不听从
- 闻秦孝公求贤，携带**李悝《法经》**西行入秦
- 四次游说秦孝公，先讲帝道、王道，秦孝公昏昏欲睡
- 最后讲**霸道、强国之术**，秦孝公"语数日不厌"

### 变法（前 356、前 350 BCE 两次）

商鞅变法是中国古代**最彻底、影响最深远的改革**：

| 领域 | 改革内容 |
|------|---------|
| **政治** | 推行县制（31 县）；编定户籍；什伍连坐 |
| **军事** | **二十等爵制**：以军功授爵——"宗室非有军功，不得为属籍" |
| **经济** | "废井田，开阡陌"——承认土地私有；重农抑商 |
| **社会** | 分户令——强制成年男子分家；统一度量衡 |
| **法律** | "刑无等级"——法律面前人人平等；轻罪重罚 |

### 徙木立信
- 变法之初，为取信于民
- 在国都南门立一根三丈之木，悬赏十金让人搬至北门
- 无人敢搬，加至五十金
- 有人尝试，立刻兑现——从此**法令必行**

### 太子犯法
- 太子（后来的秦惠文王）犯法
- 商鞅不能直接处罚太子，便处罚其师傅：**公子虔割鼻、公孙贾黥面**
- "法之不行，自上犯之"——这个原则使秦法严明

## 结局

- 前 338 年，秦孝公去世，太子即位为秦惠文王
- 公子虔等旧贵族诬告商鞅谋反
- 商鞅逃亡，投宿旅店，因无证件被拒——**"作法自毙"**
- 起兵抵抗失败，被秦军杀死于彤地
- 车裂尸体示众，**灭其家族**

> **但商鞅虽死，秦法未变。** 秦国沿着商鞅开辟的道路继续前进，最终统一天下。

## 评价

- **韩非**：推崇商鞅为法家典范
- **司马迁**：批评商鞅"刻薄少恩"，但承认其变法使秦国富强
- **后世**：商鞅是中国改革史上最重要的人物之一；其"以法治国"思想影响深远

## 名言

> "治世不一道，便国不法古"——改革不必遵循旧制

> "法之不行，自上犯之"——法律失效源于上层破坏
