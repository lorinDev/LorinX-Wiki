# 齐国 (Qi)

## 概述

齐国是西周开国功臣**姜太公（姜尚/吕尚）**的封国，都**临淄**（今山东淄博）。春秋时期在[[spring-autumn-period|齐桓公]]和管仲的治理下成为**春秋首霸**，战国时为七雄之一。公元前 221 年被秦所灭。

## 地理位置

以今山东省北部为中心，东临大海，拥有鱼盐之利。临淄是当时东方最大的都市，《战国策》记载"临淄之中七万户"。

## 春秋时期：霸业之路

### 管仲改革（约前 685 BCE）

齐桓公即位后，不计前嫌任用管仲为相，推行全面改革：

| 领域 | 改革内容 |
|------|---------|
| **政治** | "叁其国而伍其鄙"——国（都城）分 21 乡，野（农村）分 5 属，加强行政管理 |
| **军事** | "作内政而寄军令"——寓兵于民，军政合一 |
| **经济** | "官山海"——盐铁国家专卖；"相地而衰征"——按土地好坏征税 |
| **外交** | **"尊王攘夷"**——借助周天子名义团结诸侯，抵抗戎狄 |

### 齐桓公称霸

- **前 679 年**：鄄之会，齐桓公首次成为公认霸主
- **前 656 年**：召陵之盟，率八国联军伐楚，迫使楚国承认周天子权威
- **前 651 年**：葵丘之会，霸业达到顶峰，周天子派使者赐胙
- 管仲和齐桓公相继去世后，齐国霸业迅速衰落，陷入内乱

## 战国时期

- 田氏代齐（前 386 年）：田氏家族取代姜姓成为国君
- 稷下学宫：齐威王、齐宣王时期建立，成为战国学术中心
- 与秦并称"东西二帝"（前 288 年）
- 前 284 年：燕将乐毅率五国联军破齐，齐国几乎亡国
- 田单复国后元气大伤，最终在前 221 年被秦灭

## 文化贡献

- 《管子》：虽托名管仲，反映齐法家思想
- 稷下学宫：汇聚[[hundred-schools-of-thought|诸子百家]]，荀子三为祭酒
- 齐地学术重实用，与鲁地重礼乐形成对比
- 阴阳五行、黄老之学与齐地渊源深厚

## 关键人物

- **姜太公**：开国君主，兵家先驱，"太公钓鱼"
- **管仲**：改革家，被诸葛亮自比
- **鲍叔牙**：以知人著称，"管鲍之交"
- **齐桓公**：春秋首霸
- **晏婴**（晏子）：春秋后期贤相，外交家
- **孙膑**：战国军事家（孙武后代），桂陵、马陵之战
